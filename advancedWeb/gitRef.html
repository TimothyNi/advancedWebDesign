Git Cheat Sheet

1. The commands you use to navigate through directories on the command line.
To change directories you have to type the following: $ cd followed by the name of the directory you want to change to. For example if you want to change to your desktop you would type $ cd desktop.
Changing drives is not something to take lightly, but it can be done using the CD much like changing a directory. You should follow the $ cd with a backslash / of the drive you are currently in. Say you are in the D drive moving to F, you'd type $ cd /D F:
To show the contents of the directory you simply have to type the following: $ ls
This lists everything in the directory
2. How to initialize a stand-alone, local Git repository.
Open up Git Bash and type git init
3. How to initialize a local Git repository from a GitHub repository.
After initializing the local Git repository, and you want to $ git add a file, and then commit it. Type $ git remote add origin https://github.com/TimothyNi/advancedWebDesign to add it to the GitHub Repository.
Type $ git push origin master to push the changes to the new repository.
4. How to get the most recent code from GitHub.
In order to get the most recent code from the repostitory on GitHub, you need to type the following into the Git Bash: $ git clone git:https://github.com/TimothyNi/advancedWebDesign
5. How to add and commit changes to a repository.
To add a file to a repository you need to have the file name, and type the following: $ git add followed by the file name.
To commit those changes type: git commit -m "Add a comment here"
6. How to check the status of the repository.
To check the status you must type the following: $ git status
7. How to save your changes to a repository out on GitHub.
You could just add the file again, and push it out to GitHub.
8. How to view a list of all the commits in a repository.
Type $ git log to view a list of all commits that have been made in a repository.
9. How to make a temporary branch from the master repository.
Type $ git rebase master
And you've made a copy of the branch on to the master repostitory.
10. How to merge a branch back into the master.
To Merge branch back in the master type: $ git merge [branch}
